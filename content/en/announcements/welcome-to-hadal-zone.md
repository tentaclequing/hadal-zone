hadal-zone is now live. This is a decentralised library for resilience, mutual aid, and skillsharing resources.

## What you can do

- **Browse**: Explore resources by category or search for specific topics.
- **Contribute**: Submit your own resources via pull request on GitHub.
- **Mirror**: Help keep the library available by pinning it on IPFS. See the [mirror page](/en/mirror/) for instructions.

## What is coming next

- More seed documents across all categories
- Expanded translations (starting with German)
- Improved search with filtering by language and tags

Stay safe, stay connected.
